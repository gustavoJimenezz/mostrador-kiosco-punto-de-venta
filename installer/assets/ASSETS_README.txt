Archivos requeridos por installer.iss
======================================

icon.ico
--------
Ícono principal del instalador y del acceso directo generado.
Especificaciones: formato ICO, múltiples resoluciones (16x16, 32x32, 48x48, 256x256).
El logo del proyecto se encuentra en la raíz: "Punto de venta logo vintage.png".
Convertirlo con: https://convertio.co/png-ico/ o con GIMP.

banner.bmp
----------
Imagen lateral del wizard de instalación.
Especificaciones: BMP 24-bit, 164 x 314 píxeles (tamaño estándar Inno Setup "modern").
Puede generarse desde el logo del proyecto escalado y con fondo blanco.
